Sistema de Reservas de Hotel Zambezi Lodge

Este projeto consiste no desenvolvimento de um website moderno e
responsivo para um sistema de reservas de hotel em Moçambique. O sistema
inclui páginas principais como Página Inicial, Login,
Registo, e outras que compõem a interface do utilizador.

###################################################################

Funcionalidades Principais

• Página inicial com design profissional inspirado na cultura de
    Moçambique\
• Formulário funcional de reservas (check-in, check-out, hóspedes)\
• Sistema de autenticação (login e registo --- frontend)\
• Layout completamente responsivo (desktop, tablet e mobile)\
• Slider automático de depoimentos\
• Animações suaves (CSS e JavaScript)\
• Modal de recuperação de senha\
• Navegação com menu adaptado para dispositivos móveis
